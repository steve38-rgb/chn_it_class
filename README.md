# 資科補充 - 教學入口網

這是一個專為資訊科技課堂設計的入口網站，旨在提供學生快速存取常用連結、了解課堂規定以及掌握最新資訊動態。

## 🌐 網站特色
- **現代化設計**：高品質視覺效果，支援響應式佈局（手機/平板/電腦）。
- **快速連結**：集中收錄 Google Workspace, 學務系統, OpenID 等常用平台。
- **直覺導航**：分類清晰的課堂規定與活動資訊。

## 🚀 部署到 GitHub Pages
1. 在 GitHub 上建立一個新的儲存庫 (Repository)。
2. 將此目錄下的所有檔案上傳至儲存庫。
3. 進入儲存庫的 **Settings > Pages**。
4. 在 **Build and deployment** 下選擇 `Deploy from a branch`。
5. 選擇 `main` 分支並點擊 **Save**。
6. 您的網站將在幾分鐘後於 `https://[你的帳號].github.io/[儲存庫名稱]/` 上線。

## 📅 維護說明
- **更新內容**：直接編輯 `index.html` 中的對應區塊即可。
- **調整樣式**：修改 `style.css` 中的 CSS 變數或樣式規則。
